
### 1. Install docker

https://docs.docker.com/docker-for-mac/install/

### 2. Install homebrew:

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```

### 3. Install socat and Xquartz:

```
brew install socat
brew cask install xquartz
```

### 4. Run the program:

```
./run.sh
```
